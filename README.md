# 密码管理器浏览器扩展

一个基于 Manifest V3 的本地加密密码管理器，支持自动记录网页登录、主密码加密存储与备注管理。适用于 Edge、Chrome 等 Chromium 内核浏览器。

## ✨ 功能

- 🔍 **自动识别登录**：监听网页表单提交，智能捕获用户名与密码
- 🔐 **主密码加密**：使用 PBKDF2 + AES-256-GCM 对所有密码进行本地加密存储
- 📝 **备注管理**：支持为每条记录添加自定义备注
- 📋 **密码列表**：解锁后查看、搜索、复制、编辑或删除已存账号
- 🧩 **高兼容性**：穿透 Shadow DOM，支持 SPA 动态渲染页面
- 🚫 **零网络传输**：所有数据仅存于浏览器本地，无需网络权限

## 🛠️ 技术栈

- HTML / CSS / JavaScript (原生)
- Chrome Extension API (Manifest V3)
- Web Crypto API (PBKDF2、AES-GCM)
- chrome.storage.local
- MutationObserver、Shadow DOM 遍历

## 🚀 安装使用

1. 下载本仓库代码到本地
2. 打开 Edge 浏览器，访问 `edge://extensions/`
3. 开启“开发人员模式”，点击“加载解压缩的扩展”
4. 选择本项目所在的文件夹
5. 首次点击扩展图标，设置一个主密码并牢记（忘记无法找回）
6. 在任意网站登录后，扩展会自动保存；手动添加或备注请在解锁后操作

## ⚠️ 安全声明

- 所有密码均使用主密码通过 Web Crypto API 进行 AES-256-GCM 加密，密钥仅存于内存，关闭扩展后即销毁
- 主密码通过 SHA-256 哈希验证，不存储明文
- **忘记主密码将导致所有数据永久无法恢复**，请妥善保管
- 本扩展**不上传任何数据至任何服务器**，请放心使用
